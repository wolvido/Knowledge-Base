#jQuery 
keywords:
	set an element css
	get an element css
	modify the css of an element

to set and get an elements css  
  
set  
```css
.css(property,value)  
```
  
get  
```css
.css(property)  
```
  
to set multiple:  
```js
$('element').css({"propertyname":"value","propertyname":"value",...});  
```
  
note: camelcase can be used in js  
instead of border-color: xx its borderColor:xx
