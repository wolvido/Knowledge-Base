#css 
border wont show unless you add the border style
```css
/* color */  
border: black;  

/* style */  
border: solid;  

/* width | style */  
border: 2px solid;  

/* style | color */  
border: solid #f33;  

/* width | style | color */  
border: 2px solid #f33;  

/* Global values */  
border: inherit;  
border: initial;  
border: revert;  
border: revert-layer;  
border: unset;

```
