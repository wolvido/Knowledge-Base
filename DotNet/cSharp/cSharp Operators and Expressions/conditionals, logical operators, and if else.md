#cSharp 

Less than: a < b
Less than or equal to: a <= b
Greater than: a > b
Greater than or equal to: a >= b
Equal to a == b
Not Equal to: a != b

and: &&
or: ||
not: !

```csharp
if (condition) 
{
     code
}
else
{
     code
}
```
else if is also exists

syntactic sugar for else if :
```csharp
var result = (condition) ? ifCode : elseCode; 
```
result will equal to ifCode or elseCode.
